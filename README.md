## 📌 Project Overview

JoblyNest is a web-based job and task marketplace developed as a Final Year Project (FYP).  
It connects job posters and job seekers through a secure, role-based system.  
The platform enables task posting, job acceptance, admin verification, and secure online payments.

---

## ❗ Problem Statement

Traditional job platforms often lack:

- Proper user verification  
- Secure role-based access control  
- Safe payment mechanisms  
- Efficient short-term task matching  

JoblyNest addresses these issues by providing an admin-controlled verification system, secure authentication, and integrated payment support.

---

## 🎯 Objectives

- Build a secure platform for job posting and hiring  
- Implement role-based access control  
- Enable skill and location-based job matching  
- Integrate secure online payments  
- Maintain structured and reliable database management  

---

## 👥 User Roles

### 🧑‍💼 Job Poster
- Create and manage job postings  
- Hire job seekers  
- Make secure payments  
- Track task status  

### 🧑‍🔧 Job Seeker
- Create and manage profile  
- Add skills and availability  
- Accept and complete jobs  

### 🛡 Admin
- Approve and verify users  
- Monitor platform activity  
- Handle complaints and disputes  

---

## 🚀 Key Features

- 🔐 Role-based authentication  
- ✔️ Admin approval system  
- 📌 Job posting and acceptance  
- 📍 Skill and location-based matching  
- 📅 Availability management  
- 💳 Secure payments via Stripe  
- 📱 Responsive user interface  

---

## 🛠 Technology Stack

| Layer | Technology |
|--------|------------|
| Frontend | React.js |
| Backend | Node.js, Express.js |
| Database & Authentication | Supabase (PostgreSQL) |
| Payments | Stripe |

---

## 🏗 System Architecture

```
React Frontend
        ↓
Node.js / Express Backend
        ↓
Supabase (PostgreSQL Database & Auth)
        ↓
Stripe Payment Gateway
```

## ✅ Conclusion

JoblyNest demonstrates a complete full-stack web application with secure authentication, role-based access control, and online payment integration.

The project fulfills the objectives of a Final Year Project and provides a scalable foundation for future enhancements.
